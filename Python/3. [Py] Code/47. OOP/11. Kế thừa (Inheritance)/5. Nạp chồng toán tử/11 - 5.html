Một số hàm đặc biệt¶

C rt nhiu hm c bit trong Python v mt trong l hm init (), hm ny c gi bt c khi no khi to mt i tng, mt bin mi trong class.

Mc ch khi s dng cc hm c bit ny l gip nhng hm ca chng ta tng thch vi cc hm c dng sn trong Python.

Np chng ton t '+'

np chng ton t ' + ', ta s s dng hm add () trong class. Ta c th trin khai nhiu cng vic bng hm ny, v d nh cng hai im ta v d bn trn.

In [8]:
class Point:
     def __init__(self, x = 0, y = 0):
        self.x = x
        self.y = y

     def __str__(self):
        return "({0},{1})".format(self.x,self.y)

     def __add__(self,other):
        x = self.x + other.x
        y = self.y + other.y
        return Point(x,y)

Np chng ton t so snh:

Python khng ch gii hn c php np chng cc ton t ton hc, m cn cho php ngi dng np chng ton t so snh.

C nhiu ton t so snh c h tr bi Python, v d nh: <, >, <=, >=, ==,...

S dng np chng ton t ny khi mun so snh cc i tng trong lp vi nhau.

V d mun so snh cc im trong class Point, hy so snh ln ca cc im ny bt u t gc ta , thc hin nh sau:

class Point:
     def __init__(self, x = 0, y = 0):
        self.x = x
        self.y = y

     def __str__(self):
        return "({0},{1})".format(self.x,self.y)

     def __lt__(self,other):
        self_mag = (self.x ** 2) + (self.y ** 2)
        other_mag = (other.x ** 2) + (other.y ** 2)
        return self_mag < other_mag
