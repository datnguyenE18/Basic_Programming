Iterator in Python¶

Iterator l cc i tng cho php ta ly tng phn t ca n, hnh ng ny c th c lp i lp li.

Iterator trong Python là gì?¶

Iterator khp mi ni trong Python, bn trong cc vng lp, comprehension, generator

N n gin ch l cc i tng cho php ta ly tng phn t, bt c khi no bn s dng vng lp hay cc k thut c c gi tr mt nhm phn t mt thi im no .

V mt k thut, Iterator trong Python phi thc hin hai phng thc c bit l iter() v next(), gi chung l giao thc iterator (Iterator Protocol)

  • Phng thc iter tr v chnh i tng iterator. Phng thc ny c yu cu ci t cho c i tng "iterable" v iterator c th s dng cc cu lnh for v in.

  • Phng thc next tr v phn t tip theo. Nu khng cn phn t no na th s c li StopIteration xy ra.

Cách vòng lặp hoạt động¶

Nh chng ta thy trong v d trn, vng lp for c th lp li t ng thng qua vic s dng danh sch. Trong thc t, vng lp for c th lp li trn bt k iterable no. Chng ta hy xem k hn v cch vng lp for c thc hin trong Python.

In [8]:

        print(element) # do something with element

c thc hin tng t nh:

In [9]:
my_list = [4, 7, 0, 3]

# iter_obj l mt iterator object to t my_list
iter_obj = iter(my_list)
 
# vng lp
while True:
     try:
        # s dng next
        print (next(iter_obj))
     except StopIteration:
        # nu xy ra li StopIteration th vng lp s c break ra ngoi
        break

0

v d ny, bn trong vng lp for ta to mt iterator object tn l iter_obj bng cch gi iter() trn my_list.

V nh bn thy th vng lp for y chnh l vng lp while v hn. Next() bn trong vng lp ly ra cc phn t thc hin cc cu lnh for loop. n khi ly ht cc gi tr th ngoi l StopIteration s c sinh ra v vng lp kt thc.

Xây dựng trình vòng lặp Iterator trong Python¶

Chng ta c th t xy dng iterator l mt class. Xy dng Iterator rt d dng trong Python, chng ta ch cn thc hin cc phng thc iter() v next().

In [10]:
class PowTwo:
 
     def __init__(self, max = 0):
        self.max = max
 
     def __iter__(self):
        self.n = 0
        return self
 
     def __next__(self):
        if self.n <= self.max:
          result = 2 ** self.n
          self.n += 1
          return result
        else:
            raise StopIteration

Phng thc iter s lm i tng tr thnh i tng iterable.

Gi tr tr v ca iter l mt iterator. N cn c phng thc next v tr v StopIteration nu khng cn phn th no na.

Iterator lặp vô hạn trong Python¶

Khng phi tt c cc i tng iterator u s c gi ht cc phn t v kt thc khi khng cn phn t. C mt s trng hp iterator s lp v hn. V d nh sau:

In [12]:
int()
# 0

inf = iter(int,1)
print (next(inf))
# 0

print (next(inf))
# 0

0
0

Chng ta c th thy rng hm int() lun tr v 0. V vy, vic truyn n di dng iter(int, 1) s tr v mt iterator lp cho n khi gi tr tr v bng 1. iu ny khng bao gi xy ra v y chnh l iterator lp v hn. Bn cn ch khi x l trong cc trng hp th ny.

Ngoi ra, bn cng c th t xy dng iterator lp v hn. V d sau y s lp v hn v tr v cc s l v khng c iu kin dng no c.

In [13]:
class InfIter:

     def __iter__(self):
        self.num = 1
        return self

     def __next__(self):
        num = self.num
        self.num += 2
        return num
